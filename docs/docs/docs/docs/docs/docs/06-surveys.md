# Surveys

An online survey was conducted among students and community members to gather requirements for the Smart Blood Donation Emergency Platform.

## Survey Results

- 85% of respondents said they faced difficulties finding blood donors during emergencies.
- 90% preferred a digital platform for blood requests.
- 78% wanted location-based donor searching.
- 88% supported receiving notifications for urgent blood requests.

The survey results confirmed the need for an efficient blood donation management system.
