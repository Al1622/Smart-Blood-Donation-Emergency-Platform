# Stakeholder Analysis

## Primary Stakeholders
1. Blood Donors
   - Register and update donor information.
   - Respond to blood requests.

2. Blood Recipients
   - Search for donors.
   - Request blood during emergencies.

3. Administrators
   - Manage users and blood requests.
   - Monitor overall system activities.

## Secondary Stakeholders
1. Hospitals
   - Refer patients to the platform.

2. NGOs and Blood Donation Organizations
   - Promote blood donation campaigns.
   - Help manage emergency blood requests.
